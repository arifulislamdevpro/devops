GIT
sudo apt update
sudo apt install git -y
git --version

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

sudo mkdir -p /var/www/your-project
sudo chown -R $USER:www-data /var/www/your-project
sudo chmod -R 775 /var/www/your-project

cd /var/www/your-project
git clone https://github.com/yourname/your-repo.git .



✅ Step 5: If You Face SSH Permission Errors
Use SSH keys:


ssh-keygen -t ed25519 -C "your_email@example.com"
Copy the public key to GitHub:


cat ~/.ssh/id_ed25519.pub
Then go to GitHub → Settings → SSH and GPG Keys → New SSH Key.

To clone with SSH:

git clone git@github.com:yourname/your-repo.git

✅ Test the connection
Run:


ssh -T git@github.com
You should see something like:

Hi yourusername! You've successfully authenticated, but GitHub does not provide shell access.
